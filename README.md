# Microservice Project using Git & GitHub

## Objective
To manage a microservices project using Git and GitHub with proper version control.

## Features Implemented
- Git repository initialization
- .gitignore setup
- Commit tracking
- Feature branching
- Merging branches

## Workflow
1. Created project structure
2. Initialized Git
3. Pushed code to GitHub
4. Created feature branch (user-service)
5. Developed feature and committed changes
6. Merged into main branch

## Tools Used
- Git
- GitHub
- VS Code
